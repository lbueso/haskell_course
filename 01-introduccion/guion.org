*** Instalación

- Debian/Ubuntu
#+BEGIN_SRC bash
sudo apt-get update
sudo apt-get install ghc
#+END_SRC

- Arch
#+BEGIN_SRC bash
sudo pacman -S ghc
#+END_SRC

- macOS
 - Instalación con homebrew
#+END_SRC bash
brew install ghc
#+END_SRC

 - instalación desde la página web
   https://www.haskell.org/platform/mac.html

- Windows
  https://www.haskell.org/platform/windows.html
